# Калькулятор (Vue 3)

Простой веб-калькулятор на Vue 3 с историей вычислений.

## Возможности

- **Базовые операции:** сложение (+), вычитание (−), умножение (*), деление (/)
- **Ввод:** цифры, десятичная точка, скобки
- **История:** список последних выражений и результатов
- **Очистка:** кнопка «C» сбрасывает текущее выражение

## Стек

- [Vue 3](https://vuejs.org/) (Composition API, CDN)
- [Tailwind CSS](https://tailwindcss.com/) (CDN) — стили

## Запуск

Файлы статические, сервер не нужен:

1. Клонируйте репозиторий и откройте папку:
   ```bash
   cd calculator-1
   ```
2. Откройте `index.html` в браузере (двойной клик или «Open with»).

Либо поднимите локальный сервер в корне проекта, например:

```bash
npx serve .
# или
python3 -m http.server 8000
```

После этого откройте в браузере указанный адрес (например, http://localhost:3000 или http://localhost:8000).

## Безопасность

Ввод перед вычислением проверяется: допускаются только цифры, пробелы и символы `+ - * / ( ) .`. Это снижает риск выполнения произвольного кода при вводе в поле выражения.
